Clinical Trials Reporting Dashboard

An interactive dashboard for investigating site reporting

How to use:

To use this dashboard first download the Record Information Download and Review History Download csv files from ClinicalTrials.gov PRS.

Under the "Records" dropdown tab select the following:

Record Information Download
ensure all fields are selected, and group selected is [All]

Review History Download
ensure group selected is [All]


Next, upload the Record Information and Review History csv files on the right of this page

Important: after uploading, wait until the top loading bar dissapears and you see plots before interacting with dashboard functions

Click the "Application Tour" button at the top right to read more about the dashboard functions

Examine the records:

Use the selectors on the left to:

  • Specify date range
  • Change the record date aggregation (monthly, quarterly, half-year)
  • Select data of interest to view

Download the reports:

  • Save the tables from registration and results
    • This will include selected dates and data, as shown in the dashboard
  • Show expected updates and results in next quarter (3 months) or next half-year (6 months)
    • The excel file will include record information and contacts

Known issues:

    Date aggregations not working as expected. We are investigating. Current aggregation set to month only.
